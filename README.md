# Food Recipe App
Mobile App With Kotlin
Food Recipe Collection App adalah aplikasi yang berfungsi untuk menyimpan dan
mengelola berbagai macam resep masakan. Resep yang disimpan dapat dilengkapi dengan
nama resep, kategori, bahan-bahan, langkah-langkah memasak, serta gambar masakan. Tujuan
dari aplikasi ini adalah untuk membantu pengguna dalam mengatur dan mendokumentasikan
resep-resep favorit mereka agar mudah diakses kembali di kemudian hari. Dengan tampilan
berbentuk kartu, pengguna dapat melihat gambar dan kategori resep secara ringkas. Saat kartu
diklik, akan muncul tampilan detail yang menampilkan isi lengkap dari resep tersebut.

# Fitur
- Login, Register, dan CRUD Profile
- CRUD Kategori
- CRUD Resep
- Sorting dan Searching Resep
- Menampilkan Resep dari suatu pengguna
- Menampilkan Detail dari Resep suatu pengguna
- Mengurutkan Resep Berdasarkan Review Terbanyak
